# How to Get Your Free TCGplayer API Key

## Step 1: Sign Up for TCGplayer
1. Go to: https://www.tcgplayer.com/
2. Click "Sign Up" or "Register" in the top right
3. Create your free account
4. Verify your email if they ask

## Step 2: Apply for API Access
1. Go to: https://www.tcgplayer.com/
2. Scroll to the bottom and click "API" or go to https://www.tcgplayer.com/api
3. Read about their API
4. Look for "Apply for API Access" or "Get Started"
5. Fill out the application:
   - Your name
   - Email
   - What you're building: "Pokemon card price tracker for personal use"
   - It's FREE for personal/educational use!

## Step 3: Get Your API Credentials
Once approved:
1. Go to your account settings
2. Find "API Settings" or "Developer Settings"
3. You'll get:
   - API Key (looks like: abc123def456...)
   - Possibly a Secret Key
